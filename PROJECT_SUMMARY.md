### 🎨 Design System

**Colors Used:**
- Primary: #FF1F3D (Red/Pink)
- Background: #0a0e27 (Navy)
- Secondary: #1a1f3a (Dark Slate)
- Foreground: #f5f5f5 (Off-white)
- Muted: #9ca3af (Gray)

**Typography:**
- Headings: System sans-serif (bold weights)
- Body: -apple-system, BlinkMacSystemFont, Roboto
- Font sizes: 12px - 64px scale

**Spacing:**
- Base unit: 4px (Tailwind standard)
- Gap classes for consistent spacing
- Responsive padding and margins
